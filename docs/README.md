## 文档约定

- 全部使用 Markdown，UTF-8，LF 行尾。
- 标题层级最多到 H4；H1 仅顶部标题。
- 命令、路径、配置项统一用 `inline code`。
- 表格用于配置项 / 状态码 / 文件清单；流程图用 ASCII 而非外部图片以便随仓库 diff。
- 引用其他文档使用相对路径：`[xxx](./12-后端开发指南.md#3-配置与环境变量)`。
- 任何代码改动必须配套修改文档，反之亦然。

- 每个文档顶部带「文档版本 + 日期 + 状态」。
